# ToDo List Manager Backend

## Description 
This project implements the api services required for to list manager. Services like adding of a task, updating task, viewing all the tasks and deleting a task.

## Features

* Implemented api service to add a task

* Implemented api service to get all tasks data

* Implemented api services to update a task

* Implemented api service to delete task
    
## Tech Stack

1. Node JS (Version :22.16.0)

2. Typescript

3. Express for backend

4. Firebase-firestore for database

## Pre-Requisite

1. Create a project in firebase console and create a database for that project.

2. Generate your sevice account keys and use your credentials while configuring firebase. 

## Installation

1. Clone the repository 

        git clone https://github.com/NivedithaGunapati-Everest/todo-list-backend.git

2. Navigate to the project directory

        todo-list-backend

3. Install the dependencies
        Make sure Node.js is installed if not 
    * To install node for Javascript: [Click Here](https://nodejs.org/en)

4. To install the required packages: 

        npm install

5. Here Jest is used for testing. To run tests use :

        npm test

6. To compile the application

        npm run build

## Usage

1. After compiling run the application using

        npm start
